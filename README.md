## Tech Stack

- **Next.js**
- **React**
- **Tailwind CSS v4**
- **Framer Motion**
- **Aceternity UI**

## Getting Started

### Installation

Install the dependencies:

```bash
npm install
```

### Development

Start the development server with HMR:

```bash
npm run dev
```

Your application will be available at [http://localhost:3000](http://localhost:3000).
